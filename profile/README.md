## ⚙️ How We Operate

- **1 human + 1 AI** — decisions fast, no meetings, no middle management
- **One Active Slot** — focus until it ships and earns
- **Dogfood everything** — we run on our own infra ([openclaw-host](https://github.com/wooogy-hq/openclaw-host), k3s + Flux)
- **Capital as tool, not master** — no VC dependency, revenue-first
